1. **Tests/TestRunner.cs** -- Create the C# test runner script following the Chickensoft GameDemo pattern exactly:

   ```csharp
   namespace OrbitalRings.Tests;

   using Godot;

   #if RUN_TESTS
   using System.Reflection;
   using GoDotTest;
   #endif

   public partial class TestRunner : Node2D
   {
   #if RUN_TESTS
       public TestEnvironment Environment = default!;
   #endif

       public override void _Ready()
       {
   #if RUN_TESTS
           Environment = TestEnvironment.From(OS.GetCmdlineArgs());
           if (Environment.ShouldRunTests)
           {
               CallDeferred(nameof(RunTests));
               return;
           }
   #endif
           // If not running tests, just quit (this scene has no game purpose)
           GetTree().Quit();
       }

   #if RUN_TESTS
       private void RunTests()
           => _ = GoTest.RunTests(Assembly.GetExecutingAssembly(), this, Environment);
   #endif
   }
   ```

   Key points:
   - All GoDotTest references are behind `#if RUN_TESTS` so the file compiles even in release builds (though it would be excluded by Compile Remove, the preprocessor guard is defense-in-depth)
   - Uses `CallDeferred(nameof(RunTests))` to ensure scene tree is fully ready before test execution starts (Pitfall 5 from research)
   - Quits if launched without --run-tests flag (this scene has no game purpose)

2. **Tests/TestRunner.tscn** -- Create a minimal Godot scene file (hand-written, not generated by editor):
