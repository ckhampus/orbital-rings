Add room visit logic to the existing CitizenNode class created in Plan 01.

**1. Add visit constants:**
- `private const float VisitTimerMin = 20.0f;` (seconds, per user decision)
- `private const float VisitTimerMax = 40.0f;` (seconds)
- `private const float VisitDurationMin = 4.0f;` (seconds inside room)
- `private const float VisitDurationMax = 8.0f;`
- `private const float DriftDuration = 0.5f;` (seconds, walkway to edge)
- `private const float FadeDuration = 0.3f;` (seconds)
- `private const float InnerDriftRadius = 3.5f;` (drift target for inner rooms)
- `private const float OuterDriftRadius = 5.5f;` (drift target for outer rooms)
- `private const float VisitProximityThreshold = 1.5f;` (in segment arcs — citizen visits room within ~1.5 segment widths)

**2. Add visit timer as a child Timer node:**
- In Initialize() (or _Ready()): create a Timer child node, one-shot=false, connect Timeout to OnVisitTimerTimeout
- Set random initial WaitTime between VisitTimerMin and VisitTimerMax
- Call Timer.Start() explicitly (not Autostart — project pattern from Phase 3 decision)

**3. Implement OnVisitTimerTimeout():**
- If _isVisiting, return early
- Need a reference to the SegmentGrid to check occupancy. Get it via: find the RingVisual node in the scene tree (use GetTree().Root.FindChild("RingVisual", true, false) or traverse), then access its Grid property. Cache this reference on first use.
  - Alternative: pass SegmentGrid reference via Initialize() from CitizenManager. This is cleaner — CitizenManager can look up RingVisual.Grid once and pass to all citizens.
  - Use the alternative: add `private SegmentGrid _grid;` field, pass in Initialize().
- Iterate all 24 segments (both rows), find the nearest occupied segment by angular distance:
  ```csharp
  float myAngle = _currentAngle;
  int bestSegment = -1;
  float bestAngleDist = float.MaxValue;
  SegmentRow bestRow = SegmentRow.Outer;

  for (int i = 0; i < SegmentGrid.TotalSegments; i++)
  {
    var (row, pos) = SegmentGrid.FromIndex(i);
    if (!_grid.IsOccupied(row, pos)) continue;
    float segMidAngle = SegmentGrid.GetStartAngle(pos) + SegmentGrid.SegmentArc * 0.5f;
    float angleDist = AngleDistance(myAngle, segMidAngle);
    if (angleDist < bestAngleDist)
    {
      bestAngleDist = angleDist;
      bestSegment = i;
      bestRow = row;
    }
  }
  ```
- If bestSegment >= 0 AND bestAngleDist < SegmentGrid.SegmentArc * VisitProximityThreshold: call StartVisit(bestRow)
- Reset timer: `_visitTimer.WaitTime = VisitTimerMin + GD.Randf() * (VisitTimerMax - VisitTimerMin); _visitTimer.Start();`

**4. Implement StartVisit(SegmentRow targetRow):**
- Set _isVisiting = true (stops _Process walking)
- Kill any active tween: `_activeTween?.Kill();`
- Determine target radius: `float targetRadius = targetRow == SegmentRow.Outer ? OuterDriftRadius : InnerDriftRadius;`
- Determine random visit duration: `float visitDuration = VisitDurationMin + GD.Randf() * (VisitDurationMax - VisitDurationMin);`
- Set up material for transparency: For each MeshInstance3D in _meshContainer, set MaterialOverride.Transparency = BaseMaterial3D.TransparencyEnum.Alpha and DepthDrawMode = BaseMaterial3D.DepthDrawModeEnum.Always (pitfall #4 prevention)
- Create chained tween (see research Pattern 2):
  ```
  Phase 1: Drift radially (DriftDuration) — lerp radius from WalkwayRadius to targetRadius
  Phase 2: Fade out (FadeDuration) — lerp all material alpha from 1.0 to 0.0
  Phase 3: Hide (Visible = false), emit CitizenEnteredRoom
  Phase 4: Wait visitDuration (TweenInterval)
  Phase 5: Show (Visible = true), set position at targetRadius, emit CitizenExitedRoom
  Phase 6: Fade in (FadeDuration) — lerp alpha from 0.0 to 1.0
  Phase 7: Drift back (DriftDuration) — lerp radius from targetRadius to WalkwayRadius
  Phase 8: Restore opaque materials (Transparency = Disabled), set _isVisiting = false
  ```
- For radial position lerp, create a helper: `private void SetRadialPosition(float angle, float radius)` that computes `Position = new Vector3(Cos(angle)*radius, surfaceY, Sin(angle)*radius)`
- For alpha lerp, create: `private void SetMeshAlpha(float alpha)` that iterates _meshContainer children and sets each MaterialOverride.AlbedoColor with modified alpha, AND for the band mesh too

**5. Add static helper:**
- `private static float AngleDistance(float a, float b)` — returns shortest angular distance: `float diff = Mathf.Abs(a - b); return Mathf.Min(diff, Mathf.Tau - diff);`

**6. Update Initialize() signature** to accept SegmentGrid: `public void Initialize(CitizenData data, float startAngle, SegmentGrid grid)`
- Store _grid = grid
- Update CitizenManager.SpawnStarterCitizens to pass grid reference

**IMPORTANT PITFALLS TO AVOID:**
- Pitfall #4: Set Transparency=Alpha AND DepthDrawMode=Always before fading. Restore Transparency=Disabled after fade-in completes.
- Pitfall #7: Always kill _activeTween before creating a new one.
- Pitfall #6: _isVisiting=true prevents click detection (Plan 02 will check this).
