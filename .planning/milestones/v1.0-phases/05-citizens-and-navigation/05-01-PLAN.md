**1. Create Scripts/Citizens/CitizenNode.cs** (namespace OrbitalRings.Citizens):
- Extends Node3D (NOT SafeNode — SafeNode extends Node, but CitizenNode needs Node3D for Position/Rotation. Instead, implement the same subscribe/unsubscribe pattern manually by overriding _EnterTree and _ExitTree with SubscribeEvents/UnsubscribeEvents virtual methods, OR extend Node3D and manually do the lifecycle).
  - Actually, check SafeNode — it extends Node, not Node3D. CitizenNode needs Node3D. Create a SafeNode3D variant or handle lifecycle manually. Simplest approach: extend Node3D, manually call subscribe/unsubscribe in _EnterTree/_ExitTree following the exact SafeNode pattern. Document why not using SafeNode (Node vs Node3D).

- **Fields:**
  - `private CitizenData _data;`
  - `private float _currentAngle;` (radians, 0 to Tau)
  - `private float _direction;` (+1.0 CCW or -1.0 CW)
  - `private float _speed;` (radians/sec, ~0.15 +/- 15%)
  - `private float _bobPhase;` (randomized per citizen to avoid sync)
  - `private bool _isVisiting = false;` (set in Plan 02)
  - `private Node3D _meshContainer;` (holds the capsule meshes from CitizenAppearance)
  - `private Tween _activeTween;` (for visit animation in Plan 02, kill-before-create pattern)

- **Constants:**
  - `public const float WalkwayRadius = 4.5f;` (midpoint of InnerRowOuter=4.0 and OuterRowInner=5.0)
  - `private const float BaseSpeed = 0.15f;` (rad/s, ~42s for full loop)
  - `private const float SpeedVariation = 0.15f;` (+-15%)
  - `private const float BobAmplitude = 0.015f;`
  - `private const float BobFrequency = 8.0f;` (rad/s, ~1.3 Hz)

- **Properties:**
  - `public CitizenData Data => _data;`
  - `public float CurrentAngle => _currentAngle;`
  - `public bool IsVisiting => _isVisiting;`

- **Initialize(CitizenData data, float startAngle):**
  - Store _data, _currentAngle = startAngle
  - _direction = GD.Randf() > 0.5f ? 1.0f : -1.0f (random CW/CCW)
  - _speed = BaseSpeed * (1.0f + (GD.Randf() * 2f - 1f) * SpeedVariation) (random +/-15%)
  - _bobPhase = GD.Randf() * Mathf.Tau (random phase to prevent sync — pitfall #5)
  - Create mesh: _meshContainer = CitizenAppearance.CreateCitizenMesh(data); AddChild(_meshContainer)
  - Compute initial position from angle

- **_Process(double delta):**
  - If _isVisiting, return (don't walk during visits)
  - Advance _currentAngle: `_currentAngle += _direction * _speed * (float)delta`
  - Wrap to [0, Tau): `if (_currentAngle < 0) _currentAngle += Mathf.Tau; if (_currentAngle >= Mathf.Tau) _currentAngle -= Mathf.Tau;`
  - Compute position: `float x = Mathf.Cos(_currentAngle) * WalkwayRadius; float z = Mathf.Sin(_currentAngle) * WalkwayRadius;`
  - Compute bob: `float bob = Mathf.Sin(_bobPhase) * BobAmplitude; _bobPhase += BobFrequency * (float)delta;`
  - Compute surface Y: `float surfaceY = SegmentGrid.RingHeight / 2f - SegmentGrid.WalkwayRecess + CitizenAppearance.GetCapsuleHeight(data.Body) / 2f;` (stand ON the walkway surface)
  - Set Position: `Position = new Vector3(x, surfaceY + bob, z);`
  - Face direction of travel (tangent to circle): `float facingAngle = _currentAngle + (_direction > 0 ? Mathf.Pi * 0.5f : -Mathf.Pi * 0.5f); Rotation = new Vector3(0, -facingAngle, 0);`

- **Public helper for Plan 02:** `public MeshInstance3D GetPrimaryMesh()` returns the first MeshInstance3D child of _meshContainer (for material access during emission glow)
