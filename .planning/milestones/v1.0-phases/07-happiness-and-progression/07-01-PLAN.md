  <action>
Create Scripts/Autoloads/HappinessManager.cs as a new Autoload singleton following the exact pattern of EconomyManager/GameEvents. Namespace: OrbitalRings.Autoloads.

The class owns:
- `_happiness` float (0.0 to 1.0), property `public float Happiness => _happiness;`
- `_arrivalTimer` Timer child node (WaitTime=60.0, OneShot=false)
- `_unlockedRooms` HashSet<string> initialized with 6 starter rooms: "bunk_pod", "air_recycler", "workshop", "reading_nook", "storage_bay", "garden_nook"
- `_housingCapacity` int tracking current housing capacity (starts at 5 for starter citizens)
- Static singleton Instance pattern

Constants:
- `HappinessGainBase = 0.08f` (calibrated: 25% unlock at ~wish 4, 60% at ~wish 12, 100% at ~50 wishes)
- `ArrivalCheckInterval = 60.0f`
- `ArrivalProbabilityScale = 0.6f` (at 100% happiness, 60% chance per check)

Unlock milestones as static readonly array:
- (0.25f, new[] { "sky_loft", "craft_lab" })
- (0.60f, new[] { "star_lounge", "comm_relay" })
Track which milestones have been crossed with a `_crossedMilestoneCount` int.

_Ready():
- Set Instance = this
- Subscribe to GameEvents.WishFulfilled += OnWishFulfilled
- Subscribe to GameEvents.RoomPlaced += OnRoomPlaced (to update housing capacity)
- Subscribe to GameEvents.RoomDemolished += OnRoomDemolished (to update housing capacity)
- Create Timer child: _arrivalTimer = new Timer(), WaitTime = ArrivalCheckInterval, OneShot = false, AddChild, Timeout += OnArrivalCheck, call _arrivalTimer.Start() explicitly (not Autostart -- locked decision from Phase 3)
- Initialize housing capacity by scanning BuildManager placed rooms for Housing category (using the event-driven approach: subscribe to RoomPlaced/RoomDemolished and maintain running total, but also do initial scan in _Ready for any pre-placed rooms)

_ExitTree():
- Unsubscribe all GameEvents handlers

OnWishFulfilled(string citizenName, string wishType):
- If _happiness >= 1.0f, return (already at max, per locked decision: wishes still generate/fulfill but happiness stays at 100%)
- Calculate gain = HappinessGainBase / (1.0f + _happiness) -- diminishing returns formula
- _happiness = Mathf.Min(_happiness + gain, 1.0f)
- Call EconomyManager.Instance?.SetHappiness(_happiness)
- Fire GameEvents.Instance?.EmitHappinessChanged(_happiness)
- Call CheckUnlockMilestones()

CheckUnlockMilestones():
- Iterate UnlockMilestones starting from _crossedMilestoneCount
- For each milestone where _happiness >= threshold and not yet crossed:
  - Add room IDs to _unlockedRooms
  - Increment _crossedMilestoneCount
  - For each room ID, fire GameEvents.Instance?.EmitBlueprintUnlocked(roomId)

OnArrivalCheck():
- If _happiness <= 0f, return (no arrivals at 0 happiness)
- Get currentPop = CitizenManager.Instance?.CitizenCount ?? 0
- If currentPop >= _housingCapacity, return (population cap reached)
- Roll: float chance = _happiness * ArrivalProbabilityScale
- If GD.Randf() < chance, call CitizenManager.Instance?.SpawnCitizen()

OnRoomPlaced(string roomType, int segmentIndex):
- Look up the room definition to check if it's Housing category
- If Housing, add its BaseCapacity to _housingCapacity
- Use BuildManager.Instance.GetPlacedRoom(segmentIndex) to get the Definition and check Category

OnRoomDemolished(int segmentIndex):
- Need to track what was there before it was demolished
- Subscribe to RoomDemolished event, but the room is already gone by the time this fires
- Better approach: subscribe to RoomPlaced to maintain a Dictionary<int, int> mapping anchorIndex to BaseCapacity for housing rooms. On RoomDemolished, look up the capacity and subtract it.
- Maintain _housingRoomCapacities Dictionary<int, int> for this tracking

CalculateHousingCapacity() public method:
- Returns _housingCapacity (useful for HUD display if needed)

IsRoomUnlocked(string roomId) public method:
- Returns _unlockedRooms.Contains(roomId)

InitializeHousingCapacity() private method called from _Ready:
- Scan all 24 segment indices via BuildManager.Instance.GetPlacedRoom(i)
- For each found room with Housing category, add BaseCapacity to _housingCapacity
- Track in _housingRoomCapacities to handle demolish

Register in project.godot:
- Add line after WishBoard entry: HappinessManager="*res://Scripts/Autoloads/HappinessManager.cs"
- This ensures HappinessManager loads LAST (after all 5 existing autoloads)

Refactor SpawnStarterCitizens to loop and call SpawnCitizen, but override the start angle for even spacing:
- Keep SpawnStarterCitizens(int count) private
- Inside the loop, create a separate private SpawnCitizenAtAngle(float angle) or just keep the loop that sets evenly spaced angles
- Alternative cleaner approach: SpawnStarterCitizens calls SpawnCitizen() in a loop, then manually repositions each citizen to even spacing. But since Initialize sets the angle, just have SpawnStarterCitizens create data and call a shared internal method.
- Simplest refactor: Extract the body of the for-loop into SpawnCitizen() but add an optional startAngle parameter:

```csharp
public CitizenNode SpawnCitizen(float? startAngle = null)
{
    var bodyTypes = System.Enum.GetValues<CitizenData.BodyType>();
    var data = new CitizenData
    {
        CitizenName = CitizenNames.GetNextName(),
        Body = bodyTypes[GD.RandRange(0, bodyTypes.Length - 1)],
        PrimaryColor = Palette[GD.RandRange(0, Palette.Length - 1)],
        SecondaryColor = Palette[GD.RandRange(0, Palette.Length - 1)]
    };
    while (data.SecondaryColor == data.PrimaryColor && Palette.Length > 1)
        data.SecondaryColor = Palette[GD.RandRange(0, Palette.Length - 1)];

    var citizen = new CitizenNode();
    float angle = startAngle ?? GD.Randf() * Mathf.Tau;
    citizen.Initialize(data, angle, _grid);
    AddChild(citizen);
    _citizens.Add(citizen);

    EconomyManager.Instance?.SetCitizenCount(_citizens.Count);
    GameEvents.Instance?.EmitCitizenArrived(data.CitizenName);
    return citizen;
}
```

Then SpawnStarterCitizens becomes:
```csharp
private void SpawnStarterCitizens(int count)
{
    for (int i = 0; i < count; i++)
    {
        float startAngle = (float)i / count * Mathf.Tau;
        SpawnCitizen(startAngle);
    }
}
```
