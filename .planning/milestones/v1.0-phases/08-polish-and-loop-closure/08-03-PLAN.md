**1. Create TitleScreen.cs** at `scripts/UI/TitleScreen.cs`:
- Namespace: `OrbitalRings.UI`
- Extends `Control` (full-screen container)
- Programmatic UI built in `_Ready()` (matches established pattern -- all UI built in code, no scene-based layout per CONTEXT.md: "clean and fast to build")

_Ready() layout:
1. Set `AnchorsPreset` to `FullRect` (covers entire viewport)
2. Create dark background: `ColorRect` covering full screen with color `new Color(0.05f, 0.05f, 0.1f, 1.0f)`
3. Title label: "Orbital Rings" centered horizontally, positioned at ~35% from top
   - Font size 48 (use `AddThemeFontSizeOverride("font_size", 48)`)
   - Color: soft warm white `new Color(0.9f, 0.85f, 0.8f)`
   - Horizontal alignment center
4. VBoxContainer centered below title (~50% from top) with button children:
   - **Continue button** (only if `SaveManager.Instance.HasSave()`):
     - Text: "Continue"
     - Minimum size: 200x50
     - Dark semi-transparent StyleBoxFlat (same pattern as MuteToggle)
     - Connect `Pressed` to `OnContinuePressed`
   - **New Station button**:
     - Text: "New Station"
     - Same styling as Continue
     - Connect `Pressed` to `OnNewStationPressed`
5. **Confirmation dialog** (initially hidden):
   - A centered PanelContainer with dark background
   - Label: "Start a new station? Your current station will be lost."
   - Two buttons: "Confirm" and "Cancel"
   - Confirm connects to `StartNewStation()`
   - Cancel hides the dialog

OnContinuePressed():
1. `var saveData = SaveManager.Instance.Load()`
2. If null: show a warning label "Save file could not be loaded." and return
3. `SaveManager.Instance.ApplyState(saveData)` -- restores Autoload state
4. `SaveManager.Instance.PendingLoad = saveData` -- stores for scene state restoration
5. `GetTree().ChangeSceneToFile("res://Scenes/QuickTest/QuickTestScene.tscn")`
6. `SaveManager.Instance.ScheduleSceneRestore()` -- triggers room/citizen/wish restoration after scene loads

OnNewStationPressed():
1. If `SaveManager.Instance.HasSave()`: show confirmation dialog
2. Else: call `StartNewStation()` directly

StartNewStation():
1. `SaveManager.Instance.ClearSave()`
2. Reset StateLoaded flags: `CitizenManager.StateLoaded = false; HappinessManager.StateLoaded = false; EconomyManager.StateLoaded = false;`
3. `GetTree().ChangeSceneToFile("res://Scenes/QuickTest/QuickTestScene.tscn")`

```csharp
public override void _Ready()
{
    // Add ambient drone (auto-plays in its own _Ready)
    var drone = new AmbientDrone();
    drone.Name = "AmbientDrone";
    AddChild(drone);

    // Add wish celebration (subscribes to WishFulfilled in its own _EnterTree)
    var celebration = new WishCelebration();
    celebration.Name = "WishCelebration";
    AddChild(celebration);

    // Add mute toggle to HUDLayer
    var hudLayer = FindChild("HUDLayer", true, false) as CanvasLayer;
    if (hudLayer != null)
    {
        var muteToggle = new MuteToggle();
        muteToggle.Name = "MuteToggle";
        hudLayer.AddChild(muteToggle);
    }
}
```
