    1. Create Scripts/UI/ClockHUD.cs — MarginContainer in namespace OrbitalRings.UI, following MoodHUD.cs structure exactly:

       **Constants:**
       - PeriodColors dictionary (Dictionary of StationPeriod to Color):
         - Morning = new Color(0.95f, 0.85f, 0.45f) — soft gold
         - Day = new Color(0.95f, 0.93f, 0.90f) — warm white
         - Evening = new Color(0.95f, 0.65f, 0.40f) — amber/coral
         - Night = new Color(0.55f, 0.70f, 0.90f) — soft blue

       **Icon helper method:**
       - GetPeriodIcon(StationPeriod) => period switch:
         - Morning => "\u2600" (sun with rays)
         - Day => "\u2600" (sun with rays)
         - Evening => "\u263D" (first quarter moon)
         - Night => "\u263D" (first quarter moon)
         - _ => "\u2600"

       **UI Nodes (private fields):**
       - HBoxContainer _hbox
       - Label _iconLabel
       - Label _periodLabel
       - Tween _popTween (kill-before-create)

       **_Ready():**
       - Create HBoxContainer with separation=4, MouseFilter=Ignore
       - Create _iconLabel: font_size=20, MouseFilter=Ignore
       - Create _periodLabel: font_size=20, MouseFilter=Ignore
       - Subscribe to GameEvents.Instance.PeriodChanged += OnPeriodChanged
       - Initialize from StationClock.Instance.CurrentPeriod (no animation on startup):
         - Set _iconLabel.Text = GetPeriodIcon(currentPeriod)
         - Set _periodLabel.Text = currentPeriod.ToString()
         - Set both label colors from PeriodColors dictionary

       **_ExitTree():**
       - Unsubscribe GameEvents.Instance.PeriodChanged -= OnPeriodChanged

       **OnPeriodChanged(StationPeriod newPeriod, StationPeriod previousPeriod):**
       - Update _iconLabel.Text = GetPeriodIcon(newPeriod)
       - Update _periodLabel.Text = newPeriod.ToString()
       - Update both label font_color from PeriodColors[newPeriod]
       - Play scale pop animation on the HBoxContainer (NOT individual labels — pop the whole widget):
         ```csharp
         _popTween?.Kill();
         _popTween = _hbox.CreateTween();
         _hbox.Scale = new Vector2(1.15f, 1.15f);
         _popTween.TweenProperty(_hbox, "scale", new Vector2(1.0f, 1.0f), 0.3f)
             .SetEase(Tween.EaseType.Out)
             .SetTrans(Tween.TransitionType.Elastic);
         ```

       Use null-safe access for GameEvents.Instance and StationClock.Instance (both may be null in test contexts).

    2. Add ClockHUD node to Scenes/QuickTest/QuickTestScene.tscn:
       - Add ext_resource for ClockHUD.cs script (use next available id)
       - Add node after MoodHUD in HUDLayer:

         anchor_right = 1.0

       - This makes StationClock the 9th autoload, initializing after all existing singletons. GameEvents is 1st, so it will be ready when StationClock._Ready() runs.
