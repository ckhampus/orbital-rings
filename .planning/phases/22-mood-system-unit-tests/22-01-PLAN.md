       - `HysteresisPreventsRapidDemotion`: Set up Cozy tier (RestoreState 0.10), then manually verify: after Update that brings mood to 0.06 (above demote threshold 0.05), tier stays Cozy. Use CalculateTier indirectly: RestoreState to set tier=Cozy at 0.10, then call Update with very small delta and lh=0 to let mood decay slightly but stay above 0.05. Alternative cleaner approach: RestoreState(0.10, 0.0) sets Cozy. Then test that Update() with a delta that would bring mood to ~0.06-0.09 keeps Cozy.        IMPORTANT pitfall: CalculateTier only steps ONE tier per call. RestoreState uses CalculateTierFromScratch (jumps to correct tier). So for hysteresis tests: use RestoreState to set the initial tier, then use Update() to move mood and let CalculateTier evaluate with hysteresis.

       Practical approach for hysteresis: RestoreState(0.10, 0.0) -> tier=Cozy. Then call Update(delta, 0) where delta causes mood to drop to target value. For mood to reach ~0.06 from 0.10: alpha = 1 - exp(-0.003662 * delta), 0.10 + (0 - 0.10)*alpha = 0.06 -> alpha = 0.4 -> delta = -ln(0.6)/0.003662 = ~139.4s. For mood to reach ~0.049: alpha = 0.51 -> delta = -ln(0.49)/0.003662 = ~194.8s. Use these computed deltas OR just use a very large delta to drop well below 0.05. The simpler approach: use a large delta (e.g., 300s) to ensure mood drops well below any demote threshold, then assert tier changed.
