Create `Scripts/Autoloads/EconomyManager.cs` following the established GameEvents Autoload pattern (static Instance, set in _Ready). Use namespace `OrbitalRings.Autoloads`.

**Core state:**
- `private int _credits` -- the credit balance (integer, no floating point)
- `private Timer _incomeTimer` -- child Timer node for periodic ticks
- `private int _citizenCount` -- registered citizen count (starts at 0, Phase 5 will call setter)
- `private int _workingCitizenCount` -- citizens in work rooms (starts at 0, Phase 5 will call setter)
- `private float _currentHappiness` -- station happiness 0.0-1.0 (starts at 0, Phase 7 will set)

**Public properties:**
- `public static EconomyManager Instance { get; private set; }`
- `[Export] public EconomyConfig Config { get; set; }`
- `public int Credits => _credits;`

**_Ready() method:**
1. `Instance = this;`
2. Load config with null-safety: `Config ??= new EconomyConfig();` with `GD.PushWarning()` if fallback used
3. Initialize `_credits = Config.StartingCredits;`
4. Create Timer as child node:
   ```
   _incomeTimer = new Timer();
   _incomeTimer.WaitTime = Config.IncomeTickInterval;
   _incomeTimer.Autostart = true;
   _incomeTimer.OneShot = false;
   AddChild(_incomeTimer);
   _incomeTimer.Timeout += OnIncomeTick;
   ```
5. Set `ProcessMode = ProcessModeEnum.Pausable` explicitly (Timer pauses with tree)
6. Emit initial CreditsChanged: `GameEvents.Instance?.EmitCreditsChanged(_credits);`

**OnIncomeTick() method -- periodic income calculation:**
```
int income = CalculateTickIncome();
if (income > 0)
{
    _credits += income;
    GameEvents.Instance?.EmitIncomeTicked(income);
    GameEvents.Instance?.EmitCreditsChanged(_credits);
}
```

**CalculateTickIncome() method -- pure calculation, public for testing:**
Income formula (per research, matching spreadsheet):
```
float citizenIncome = Config.PassiveIncomePerCitizen * MathF.Sqrt(_citizenCount);
float baseIncome = Config.BaseStationIncome;
float workBonus = _workingCitizenCount * Config.WorkBonusMultiplier;
float happinessMult = 1.0f + (_currentHappiness * (Config.HappinessMultiplierCap - 1.0f));
return Mathf.RoundToInt((baseIncome + citizenIncome + workBonus) * happinessMult);
```

**TrySpend(int amount) method -- returns bool, deducts only if affordable:**
```
if (amount <= 0 || _credits < amount) return false;
_credits -= amount;
GameEvents.Instance?.EmitCreditsSpent(amount);
GameEvents.Instance?.EmitCreditsChanged(_credits);
return true;
```

**Earn(int amount) method -- add credits (for future use):**
```
if (amount <= 0) return;
_credits += amount;
GameEvents.Instance?.EmitCreditsChanged(_credits);
```

**Refund(int amount) method -- add refund credits with event:**
```
if (amount <= 0) return;
_credits += amount;
GameEvents.Instance?.EmitCreditsRefunded(amount);
GameEvents.Instance?.EmitCreditsChanged(_credits);
```

**CalculateRoomCost(RoomDefinition room, int segmentCount, bool isOuterRow) method -- pure function:**
```
float baseCost = room.BaseCostOverride > 0 ? room.BaseCostOverride : Config.BaseRoomCost;
float catMult = GetCategoryMultiplier(room.Category);
float sizeFactor = segmentCount * MathF.Pow(Config.SizeDiscountFactor, segmentCount - 1);
float rowMult = isOuterRow ? Config.OuterRowCostMultiplier : 1.0f;
return Mathf.RoundToInt(baseCost * catMult * sizeFactor * rowMult);
```

**CalculateDemolishRefund(int originalCost) method:**
```
return Mathf.RoundToInt(originalCost * Config.DemolishRefundRatio);
```

**GetCategoryMultiplier(RoomDefinition.RoomCategory category) method -- reads from Config:**
```
return category switch
{
    RoomDefinition.RoomCategory.Housing => Config.HousingCostMultiplier,
    RoomDefinition.RoomCategory.LifeSupport => Config.LifeSupportCostMultiplier,
    RoomDefinition.RoomCategory.Work => Config.WorkCostMultiplier,
    RoomDefinition.RoomCategory.Comfort => Config.ComfortCostMultiplier,
    RoomDefinition.RoomCategory.Utility => Config.UtilityCostMultiplier,
    _ => 1.0f
};
```

**Phase 5/7 integration stubs (public setters for future use):**

```
public void SetWorkingCitizenCount(int count) { _workingCitizenCount = Mathf.Max(0, count); }
public void SetHappiness(float happiness) { _currentHappiness = Mathf.Clamp(happiness, 0f, 1f); }
```

**GetIncomeBreakdown() method -- returns components for tooltip display:**
```
public (float baseIncome, float citizenIncome, float workBonus, float happinessMult, int total) GetIncomeBreakdown()
{
    float baseIncome = Config.BaseStationIncome;
    float citizenIncome = Config.PassiveIncomePerCitizen * MathF.Sqrt(_citizenCount);
    float workBonus = _workingCitizenCount * Config.WorkBonusMultiplier;
    float happinessMult = 1.0f + (_currentHappiness * (Config.HappinessMultiplierCap - 1.0f));
    int total = Mathf.RoundToInt((baseIncome + citizenIncome + workBonus) * happinessMult);
    return (baseIncome, citizenIncome, workBonus, happinessMult, total);
}
```

Use `using System;` for MathF. Use `using Godot;` and `using OrbitalRings.Data;` for RoomDefinition and EconomyConfig. Add XML doc comments for all public methods.
