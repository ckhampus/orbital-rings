**FloatingText.cs** -- A reusable Label that drifts upward and fades out, then frees itself. Namespace `OrbitalRings.UI`.

```csharp
public partial class FloatingText : Label
{
    public void Setup(string text, Color color, Vector2 startPosition)
    {
        Text = text;
        Position = startPosition;
        AddThemeColorOverride("font_color", color);
        // Optional: set font size larger
        AddThemeFontSizeOverride("font_size", 18);

        var tween = CreateTween();
        tween.SetParallel(true);
        // Drift upward 50-60px over 0.8-1.0 seconds
        tween.TweenProperty(this, "position:y", startPosition.Y - 55f, 0.9f)
            .SetEase(Tween.EaseType.Out);
        // Fade out with slight delay so text is readable briefly
        tween.TweenProperty(this, "modulate:a", 0.0f, 0.9f)
            .SetEase(Tween.EaseType.In)
            .SetDelay(0.2f);
        tween.SetParallel(false);
        tween.TweenCallback(Callable.From(QueueFree));
    }
}
```

Colors: green for income/refund (e.g., `new Color(0.3f, 0.85f, 0.3f)`), red for spend (e.g., `new Color(0.95f, 0.3f, 0.3f)`).

**Rolling counter animation:**
- Track `_displayedCredits` (float) and `_targetCredits` (int)
- On CreditsChanged: kill existing tween (`_activeTween?.Kill()`), create new tween from `_displayedCredits` to `newBalance` over 0.4 seconds with ease-out cubic

- After tween completes, set `_displayedCredits = _targetCredits`

**Income tick flash:**
- On IncomeTicked event: briefly change the balance label color to warm gold `new Color(1.0f, 0.92f, 0.6f)` and tween back to default white over 0.5 seconds
- Also spawn a FloatingText "+{amount}" in green near the balance label position

**Spend/refund floating text:**
- On CreditsSpent: spawn FloatingText "-{amount}" in red near the balance label
- On CreditsRefunded: spawn FloatingText "+{amount}" in green near the balance label
- Floating texts are added to the CreditHUD itself (same CanvasLayer) so they render on top of 3D

**Income breakdown tooltip on hover:**
- Override `_GuiInput` or connect `MouseEntered`/`MouseExited` events
- Set `MouseFilter = MouseFilterEnum.Stop` on the HBoxContainer or CreditHUD to capture hover
- On hover: create/show a PanelContainer tooltip near the balance label showing:

  Base station: +1
  Citizens: +0 (0 citizens)
  Work bonus: +0
  Happiness: x1.00
  ───────────
  Total: +1 / tick

- Call `EconomyManager.Instance.GetIncomeBreakdown()` to get the values

- Style the tooltip similarly to SegmentTooltip (PanelContainer with StyleBox)

**Event subscription pattern (manual, matching SegmentTooltip convention):**
```csharp
public override void _EnterTree()
{
    if (GameEvents.Instance != null)
    {
        GameEvents.Instance.CreditsChanged += OnCreditsChanged;
        GameEvents.Instance.IncomeTicked += OnIncomeTicked;
        GameEvents.Instance.CreditsSpent += OnCreditsSpent;
        GameEvents.Instance.CreditsRefunded += OnCreditsRefunded;
    }
}

public override void _ExitTree()
{
    if (GameEvents.Instance != null)
    {
        GameEvents.Instance.CreditsChanged -= OnCreditsChanged;
        GameEvents.Instance.IncomeTicked -= OnIncomeTicked;
        GameEvents.Instance.CreditsSpent -= OnCreditsSpent;
        GameEvents.Instance.CreditsRefunded -= OnCreditsRefunded;
    }
}
```

Use `using Godot;` and `using OrbitalRings.Autoloads;`. Add XML doc comments for the class.
