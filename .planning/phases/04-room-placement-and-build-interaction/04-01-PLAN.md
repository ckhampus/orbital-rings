    - `public void OnSegmentClicked(int flatIndex)`:
      Switch on _mode:
      - **Normal**: no-op (let SegmentInteraction handle normal selection)
      - **Placing**:
        - If `_anchorFlatIndex < 0`: This is the first click — set anchor. Validate segment is free. If free, set `_anchorFlatIndex = flatIndex`, decompose to row/pos, set `_anchorRow`, `_startPos`, `_currentSize = 1`. Create ghost preview via RoomVisual.CreateRoomBlock with isGhost=true. Add ghost as child of the node that holds room visuals (use _ringVisual parent or a dedicated container).
        - If `_anchorFlatIndex >= 0`: This is the confirm click. Validate all segments in current selection are free AND player can afford it. If valid:
          1. Calculate cost via `EconomyManager.Instance.CalculateRoomCost`
          2. Call `EconomyManager.Instance.TrySpend(cost)` — if false, emit invalid feedback event and return
          3. Mark segments occupied via `_ringVisual.Grid.SetSegmentsOccupied`
          4. Remove ghost, create permanent room block via `RoomVisual.CreateRoomBlock(... isGhost: false)`
          5. Add to `_placedRooms` dictionary
          6. Emit `GameEvents.Instance.EmitRoomPlaced(room.RoomId, anchorFlatIndex)`
          7. Emit placement feedback event (new event on GameEvents: `RoomPlacementConfirmed` with MeshInstance3D ref and position)
          8. Reset placement state (_anchorFlatIndex = -1, _currentSize = 1, _selectedRoom stays for repeated placement)
          9. If invalid (occupied or can't afford), flash red on the segment
      - **Demolish**:
        - Find placed room at this flat index (check _placedRooms — a room at startPos occupies startPos to startPos+count-1, need to check if flatIndex falls within any placed room's range)
        - If no room found, ignore
        - If `_pendingDemolishIndex == flatIndex` (second click = confirm):
          1. Get PlacedRoom data
          2. Calculate refund via `EconomyManager.Instance.CalculateDemolishRefund(room.Cost)`
          3. Call `EconomyManager.Instance.Refund(refund)`
          4. Free segments via `_ringVisual.Grid.SetSegmentsOccupied(row, startPos, count, false)`
          5. Remove room mesh (QueueFree)
          6. Remove from _placedRooms
          7. Emit `GameEvents.Instance.EmitRoomDemolished(flatIndex)`
          8. Emit demolish feedback event
          9. Reset _pendingDemolishIndex = -1
        - Else (first click):
          Set `_pendingDemolishIndex = flatIndex`
          Emit event for UI to show "Demolish? (+N)" confirm popup

        - Check if all segments in new selection are free — if not, show invalid color on ghost
    - `_Input` handler:
