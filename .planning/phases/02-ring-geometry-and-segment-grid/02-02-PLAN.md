Create `Scripts/Ring/SegmentInteraction.cs` in namespace `OrbitalRings.Ring`. This is a `Node3D` that extends `SafeNode` (it subscribes to GameEvents).

**Fields:**
- `private RingVisual _ringVisual` -- sibling reference, obtained in _Ready via `GetParent().GetNode<RingVisual>(...)` or by expecting a specific node path
- `private Camera3D _camera` -- obtained in _Ready via `GetViewport().GetCamera3D()`
- `private SegmentTooltip _tooltip` -- set via a public setter or found by node path (see Task 2 wiring)
- `private readonly Plane _ringPlane = new(Vector3.Up, SegmentGrid.RingHeight * 0.5f)` -- horizontal plane at ring surface height
- `private int _hoveredFlatIndex = -1` -- currently hovered segment flat index, -1 = none
- `private int _selectedFlatIndex = -1` -- currently selected segment, -1 = none
- `private Vector2 _lastMousePos` -- cached mouse position for _Process recalculation

**Input handling in `_Input(InputEvent @event)`:**
- `InputEventMouseMotion`: cache `_lastMousePos = mm.Position`, call `UpdateHover()`
- `InputEventMouseButton` left-click (Pressed): If `_hoveredFlatIndex >= 0`, call `SelectSegment(_hoveredFlatIndex)`. Consume the event via `GetViewport().SetInputAsHandled()` to prevent it from propagating.
- `InputEventKey` Escape (Pressed): Call `DeselectSegment()`. Register "deselect" input action programmatically in _Ready following OrbitalCamera's EnsureInputActions pattern (check `InputMap.HasAction("deselect")`, register Key.Escape if missing). Use `Input.IsActionJustPressed("deselect")` in _Input instead of raw key check for consistency.

Actually, simpler: detect `@event is InputEventKey key && key.Pressed && key.Keycode == Key.Escape` directly. The deselect action is only used here and raw key detection is fine for a single key. Avoid over-engineering the input action registration for one key.

**`_Process(double delta)` -- hybrid hover update:**
- Always call `UpdateHover()` to handle camera orbit invalidating hover state while mouse is stationary. This is cheap (one ray-plane intersection + polar math per frame).

**`UpdateHover()` method:**
1. If `_camera` is null, return (safety)
2. Cast ray from camera through `_lastMousePos`:
   - `Vector3 rayOrigin = _camera.ProjectRayOrigin(_lastMousePos)`
   - `Vector3 rayDir = _camera.ProjectRayNormal(_lastMousePos)`
3. Intersect with `_ringPlane`:
   - `Vector3? hit = _ringPlane.IntersectsRay(rayOrigin, rayDir)`
   - If null, call `ClearHover()` and return
4. Convert to polar coordinates:
   - `float distance = Mathf.Sqrt(hit.Value.X * hit.Value.X + hit.Value.Z * hit.Value.Z)`
   - `float angle = Mathf.Atan2(hit.Value.Z, hit.Value.X)`
5. Determine row:
   - If `distance >= SegmentGrid.OuterRowInner && distance <= SegmentGrid.OuterRadius` -> isOuter = true
   - Else if `distance >= SegmentGrid.InnerRadius && distance <= SegmentGrid.InnerRowOuter` -> isOuter = false
   - Else -> `ClearHover()` and return (in walkway or outside ring)
6. Convert angle to segment index:
   - Normalize: `if (angle < 0) angle += Mathf.Tau`
   - `int segIndex = (int)(angle / SegmentGrid.SegmentArc) % SegmentGrid.SegmentsPerRow`
7. Compute flat index: `int flatIndex = SegmentGrid.ToIndex(isOuter ? SegmentRow.Outer : SegmentRow.Inner, segIndex)`
8. If flatIndex != _hoveredFlatIndex:
   - Unhover old segment (if != -1 and != _selectedFlatIndex, set to Normal)
   - Set _hoveredFlatIndex = flatIndex
   - If flatIndex != _selectedFlatIndex, set visual state to Hovered via `_ringVisual.SetSegmentState(flatIndex, SegmentVisualState.Hovered)`
   - Emit `GameEvents.Instance.EmitSegmentHovered(segIndex, isOuter)`
9. Update tooltip: get label from `_ringVisual.Grid.GetLabel(row, segIndex)`, call `_tooltip?.Show(label, _lastMousePos)`

**`ClearHover()` method:**
- If `_hoveredFlatIndex != -1`:
  - If `_hoveredFlatIndex != _selectedFlatIndex`: set visual to Normal
  - `_hoveredFlatIndex = -1`
  - `GameEvents.Instance?.EmitSegmentUnhovered()`
  - `_tooltip?.Hide()`

**`SelectSegment(int flatIndex)` method:**
- If flatIndex == _selectedFlatIndex: return (already selected, no-op)
- Deselect old: if `_selectedFlatIndex >= 0`, set old to Normal (or Hovered if it's currently hovered)
- Set `_selectedFlatIndex = flatIndex`
- Set visual: `_ringVisual.SetSegmentState(flatIndex, SegmentVisualState.Selected)`
- Decompose: `var (row, pos) = SegmentGrid.FromIndex(flatIndex)`
- Emit: `GameEvents.Instance?.EmitSegmentSelected(pos, row == SegmentRow.Outer)`
- Log: `GD.Print($"Selected: {_ringVisual.Grid.GetLabel(row, pos)}")` for debugging/verification

**`DeselectSegment()` method:**
- If `_selectedFlatIndex < 0`: return
- Set old selected to Normal (or Hovered if it happens to be the currently hovered segment)
- `_selectedFlatIndex = -1`
- `GameEvents.Instance?.EmitSegmentDeselected()`

**SubscribeEvents/UnsubscribeEvents:** Not needed unless listening for external events. For now, no subscriptions -- this node EMITS events but doesn't consume them. However, still extend SafeNode for future-proofing (subscribe/unsubscribe are no-ops).

CRITICAL NOTES:
- Do NOT use physics/collision for picking. Use Plane.IntersectsRay only.
- The angle-to-segment mapping must NOT try to align with screen orientation (camera orbits). Segments are at fixed world angles.
- When hovering a selected segment, keep it in Selected state (don't downgrade to Hovered).
