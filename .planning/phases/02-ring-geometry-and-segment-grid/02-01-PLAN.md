Create three files in `Scripts/Ring/` namespace `OrbitalRings.Ring`:

**SegmentGrid.cs** -- Pure C# data model (no Godot Node dependency). Contains:
- `SegmentRow` enum: `Outer`, `Inner`
- `SegmentGrid` class with constants: `SegmentsPerRow = 12`, `TotalSegments = 24`, `OuterRadius = 6.0f`, `InnerRadius = 3.0f`, `OuterRowInner = 5.0f`, `InnerRowOuter = 4.0f`, `RingHeight = 0.3f`, `WalkwayRecess = 0.025f`, `SegmentArc = Mathf.Tau / 12`
- `bool[] _occupied` array (24 slots)
- Methods: `IsOccupied(row, position)`, `SetOccupied(row, position, occupied)`, `ToIndex(row, position)` (static), `FromIndex(index)` (static), `GetStartAngle(position)` (static), `GetLabel(row, position)` returns "Outer 3 -- Empty" format (1-based display), `GetRowRadii(row)` returns (inner, outer) tuple
- Follow the exact code structure from the research document's "Complete Segment Grid Data Model" example

**RingColors.cs** -- Static color palette class. Contains:
- `OuterBase` = soft rose `Color(0.91f, 0.78f, 0.80f)`, `OuterAlt` = slightly deeper rose `Color(0.88f, 0.74f, 0.77f)`
- `InnerBase` = soft lavender `Color(0.82f, 0.78f, 0.91f)`, `InnerAlt` = slightly deeper lavender `Color(0.78f, 0.74f, 0.88f)`
- `Walkway` = warm beige `Color(0.88f, 0.85f, 0.78f)`
- `Brighten(Color c, float factor = 0.15f)` -- clamps R/G/B additions to 1.0
- `SelectionHighlight(Color c)` -- brighten by 0.25 + warm gold accent shift (+0.05 R, +0.03 G)
- `GetBaseColor(SegmentRow row, int position)` -- alternates base/alt by even/odd position

**RingMeshBuilder.cs** -- Static helper class. Contains:
- `public static ArrayMesh CreateAnnularSector(float innerR, float outerR, float startAngle, float endAngle, float height, int subdivisions = 4)` using SurfaceTool
- Generates: top face (CCW winding from above), bottom face (reversed winding), outer curved face, inner curved face, two radial side faces (start edge and end edge)
- Uses `st.Begin(Mesh.PrimitiveType.Triangles)`, `st.SetNormal()`, `st.AddVertex()`, `st.Commit()`
- Private helper: `AddQuad(SurfaceTool st, Vector3 a, b, c, d, Vector3 normal)` -- two triangles per quad (ABC, ACD), each with normals set before each vertex
- For side faces: use outward-facing normals (e.g., outer face normal = radial direction at that angle; inner face normal = inward radial; side face normals = perpendicular to the radial edge)
- `halfH = height * 0.5f`, top Y = halfH, bottom Y = -halfH
- Follow the SurfaceTool pattern from the research document

CRITICAL: Set normals BEFORE each AddVertex call (SurfaceTool is stateful -- SetNormal applies to the next AddVertex). Use CCW winding order for front faces.

**RingVisual.cs** -- `Node3D` in namespace `OrbitalRings.Ring`. This is NOT a SafeNode -- it is a visual builder, not a signal consumer. Contains:
- `private MeshInstance3D[] _segmentMeshes = new MeshInstance3D[SegmentGrid.TotalSegments]`
- `private StandardMaterial3D[] _baseMaterials = new StandardMaterial3D[SegmentGrid.TotalSegments]`
- `private StandardMaterial3D[] _hoverMaterials = new StandardMaterial3D[SegmentGrid.TotalSegments]`
- `private StandardMaterial3D[] _selectedMaterials = new StandardMaterial3D[SegmentGrid.TotalSegments]`
- `private SegmentGrid _grid = new SegmentGrid()`
- `public SegmentGrid Grid => _grid` (public read access for interaction system)
- `_Ready()`: calls `BuildSegments()` then `BuildWalkway()`
- `BuildSegments()`: For each of 24 segments (outer 0-11, inner 0-11):
  - Get row radii via `SegmentGrid.GetRowRadii(row)`
  - Get start angle via `SegmentGrid.GetStartAngle(position)`, end = start + SegmentGrid.SegmentArc
  - Call `RingMeshBuilder.CreateAnnularSector(innerR, outerR, startAngle, endAngle, SegmentGrid.RingHeight)`
  - Create `MeshInstance3D`, set mesh, create 3 `StandardMaterial3D` instances (base, hover, selected) using `RingColors.GetBaseColor`, `RingColors.Brighten`, `RingColors.SelectionHighlight`
  - Set `MaterialOverride` to base material
  - `AddChild(meshInstance)`, store in arrays by flat index (`SegmentGrid.ToIndex`)
- `BuildWalkway()`: Create one full annulus mesh via `RingMeshBuilder.CreateAnnularSector(SegmentGrid.InnerRowOuter, SegmentGrid.OuterRowInner, 0, Mathf.Tau, SegmentGrid.RingHeight - SegmentGrid.WalkwayRecess * 2, subdivisions: 48)` -- use 48 subdivisions for smooth full circle. Position it with `Transform = new Transform3D(Basis.Identity, new Vector3(0, -SegmentGrid.WalkwayRecess, 0))` to recess it below the row surfaces. Material: `RingColors.Walkway`, single `StandardMaterial3D`.
- `public void SetSegmentState(int flatIndex, SegmentVisualState state)`: swaps `MaterialOverride` on `_segmentMeshes[flatIndex]` between `_baseMaterials`, `_hoverMaterials`, or `_selectedMaterials`.
- `public MeshInstance3D GetSegmentMesh(int flatIndex)` accessor for future use.
- `SegmentVisualState` enum: `Normal`, `Hovered`, `Selected`
