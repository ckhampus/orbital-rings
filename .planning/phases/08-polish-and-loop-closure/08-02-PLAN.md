```csharp
// GenerateTone: sine wave with linear decay envelope
private static AudioStreamWav GenerateTone(float frequency, float duration, int sampleRate = 22050)
{
    int sampleCount = (int)(sampleRate * duration);
    byte[] data = new byte[sampleCount * 2];
    for (int i = 0; i < sampleCount; i++)
    {
        float t = (float)i / sampleRate;
        float envelope = 1.0f - (t / duration);
        float sample = MathF.Sin(2 * MathF.PI * frequency * t) * envelope * 0.5f;
        short pcm = (short)(sample * short.MaxValue);
        data[i * 2] = (byte)(pcm & 0xFF);
        data[i * 2 + 1] = (byte)((pcm >> 8) & 0xFF);
    }
    var stream = new AudioStreamWav();
    stream.Format = AudioStreamWav.FormatEnum.Format16Bits;
    stream.MixRate = sampleRate;
    stream.Data = data;
    return stream;
}
```

**AmbientDrone.cs:**
- Namespace: `OrbitalRings.Audio`
- Extends `Node` (will be added as child of main scene or an Autoload)
- Generates a procedural space drone: soft, low-frequency hum with gentle modulation

Generation details (Claude's discretion per CONTEXT.md):
- Base frequency: ~60 Hz (low hum, like distant machinery)
- Add harmonics: fundamental (0.4 amplitude) + perfect fifth at 1.5x freq (0.15) + octave at 2x freq (0.1)
- Gentle amplitude modulation: `0.85 + 0.15 * sin(2*PI*0.3*t)` for slow wobble
- Overall volume: multiply by 0.3 to keep quiet (background, not foreground)
- Buffer duration: 4 seconds at 22050 Hz sample rate
- CRITICAL for seamless loop: Calculate buffer length as exact multiple of base frequency period. Period = sampleRate / baseFreq. Use `periods = (int)(duration * baseFreq)` and `sampleCount = (int)(periods * period)` to ensure zero-crossing alignment at loop boundary.
- Set `stream.LoopMode = AudioStreamWav.LoopModeEnum.Forward`, `stream.LoopBegin = 0`, `stream.LoopEnd = sampleCount`

PlaybackSetup:
- Create an `AudioStreamPlayer` child node in `_Ready()`
- Assign the generated stream
- Set volume to -12 dB (quiet background)
- Call `Play()` in `_Ready()`

Public API:
- `public void StartPlaying()` -- plays if not already playing
- `public void StopPlaying()` -- stops playback
